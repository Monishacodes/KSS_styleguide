6.1 Footer Navigation

Consists out of components grouped together. They rely entirely on nesting, so make sure people are aware of this.

Example

  • Facebook
  • Twitter
  • Google+
  • LinkedIn
  • Behance
  • Dribble
  • GitHub 

<footer class="footer">
    <div class="container text-center">
        <ul class="footer-links p-0">
           <li>
                <a href="www.facebook.com">Facebook</a>
            </li>
            <li>
                <a href="www.twitter.com">Twitter</a>
            </li>
            <li>
                <a href="www.google.com">Google+</a>
            </li>
            <li>
                <a href="www.linkedin.com">LinkedIn</a>
            </li>
            <li>
                <a href="www.behance.com">Behance</a>
            </li>
            <li>
                <a href="www.Dribble.com">Dribble</a>
            </li>
            <li>
                <a href="www.Github.com">GitHub</a>
            </li>
        </ul>
    </div>
</footer><br>

  • ABOUT
  • WORK
  • BLOG
  • CONTACT

                    </div>
                    <div class="col-sm-9 text-right col-xs-9 pr-0">
