4.1 Input Elements

Consists out of components grouped together. They rely entirely on nesting, so make sure people are aware of this.

Input type text and textarea

Example

<section class="contact">
    <div class="contact-form">
        <form action="get" class="container">
            <div class="p-relative mt-30">
                <input type="text" name="name" required>
                <label for="">Your Name <span>*</span></label>
            </div>
            <div class="p-relative">
               <textarea name="" id="" cols="30" rows="9" class="postion-relative" required></textarea>
                <label for="">Your Message <span>*</span></label>
            </div>
        </form>
    </div>
</section>
